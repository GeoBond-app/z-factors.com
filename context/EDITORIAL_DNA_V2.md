# THREE ARTICLE TYPES

TYPE 1   FULL ANALYSIS
When: facts clear, human impact clear, response exists
Layer 1: The real story, full length
Layer 2+3: Feeling and response combined
           50% of Layer 1, revealed on click

TYPE 2   COMMENTARY
When: story needs interpretation
      questions more valuable than false certainty
Layer 1: What we know
Layer 2+3: Sharp commentary, perspective on click

TYPE 3   ONE QUESTION
When: story too early, too complex
      honest uncertainty better than manufactured answer
Format: 2-3 sentence context
        One powerful question
        Nothing else

# EDITORIAL DECISION ENGINE

Every story scored on four dimensions:

FACT CLARITY        0-10
HUMAN IMPACT        0-10
RESPONSE EXISTS     0-10
QUESTION VALUE      0-10

RULES
Never force Layer 3 if no real response exists.
Silence is better than manufactured optimism.
One honest question beats five false answers.
